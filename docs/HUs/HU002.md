# HU002 — Necesito registrar fincas y lotes

**Identificador:** HU002

## Historia de Usuario

> **Como** Administrador,  
> **quiero** registrar fincas y lotes,  
> **para** organizar la estructura productiva del sistema.

## Criterios de Aceptación

| # | Criterio | Contexto | Evento | Resultado Esperado |
|---|----------|----------|--------|--------------------|
| 1.0 | Crear, editar o eliminar fincas | Se desea agregar o gestionar fincas dentro del sistema | El administrador accede al módulo de fincas | El sistema permite crear nuevas fincas, editar las existentes o eliminarlas correctamente |
| 2.0 | Registrar lotes dentro de cada finca | Una finca ya está creada | El administrador selecciona la finca y registra un lote | El sistema crea el lote asociado correctamente |
| 3.0 | Registrar capacidad, ubicación y categoría del lote | Se necesita información detallada de un lote | El administrador ingresa los datos en el formulario | El sistema guarda la información en la ficha del lote |
| 4.0 | Validar nombres únicos por finca | Se crea un lote o finca con nombre repetido | El administrador intenta guardar el nombre | El sistema alerta que el nombre ya existe e impide la duplicación |
