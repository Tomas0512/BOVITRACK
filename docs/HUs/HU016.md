# HU016 — Quiero registrar datos sin conexión

**Identificador:** HU016

## Historia de Usuario

> **Como** usuario,  
> **quiero** registrar datos sin conexión,  
> **para** sincronizarlos cuando tenga internet.

## Criterios de Aceptación

| # | Criterio | Contexto | Evento | Resultado Esperado |
|---|----------|----------|--------|--------------------|
| 1.0 | Guardar datos localmente | No hay conexión a internet disponible | El usuario realiza un registro (sanidad, producción, etc.) | El sistema almacena temporalmente la información en el dispositivo |
| 2.0 | Sincronizar automáticamente | Se recupera la conexión | El usuario abre la app o activa la sincronización | El sistema envía los datos pendientes al servidor |
| 3.0 | Mostrar conflictos de sincronización | Existen registros que no coinciden con el servidor | El sistema detecta diferencias al sincronizar | Se notifican los conflictos y se solicita resolución al usuario |
