# HU005 — Quiero registrar alimentación, ordeños y pesajes

**Identificador:** HU005

## Historia de Usuario

> **Como** usuario autorizado,  
> **quiero** registrar alimentación, ordeños y pesajes,  
> **para** llevar control productivo del ganado..

## Criterios de Aceptación

| # | Criterio | Contexto | Evento | Resultado Esperado |
|---|----------|----------|--------|--------------------|
| 1.0 | Registrar tipo y cantidad de alimento | Se debe registrar la alimentación diaria del animal o lote | El usuario autorizado ingresa al módulo de alimentación y especifica tipo y cantidad | El sistema almacena el registro y lo asocia al animal o lote |
| 2.0 | Registrar ordeño diario por animal | Se realiza un ordeño | El usuario ingresa la cantidad ordeñada en el sistema | El sistema guarda el registro en el historial productivo del animal |
| 3.0 | Registrar pesajes con fecha | Se pesa un animal | El usuario ingresa el peso y la fecha correspondiente | El sistema almacena la información y la agrega a la evolución del animal |
| 4.0 | Ver gráficos de evolución | El animal ya tiene varios registros de alimentación, ordeño y peso | El usuario consulta la evolución productiva | El sistema genera gráficos comparativos y líneas de tiempo |
