# HU004 — Quiero registrar y administrar animales

**Identificador:** HU004

## Historia de Usuario

> **Como** Capataz,  
> **quiero** registrar y administrar animales,  
> **para** mantener la información del hato actualizado.

## Criterios de Aceptación

| # | Criterio | Contexto | Evento | Resultado Esperado |
|---|----------|----------|--------|--------------------|
| 1.0 | Registrar información general del animal | Se incorpora un animal nuevo al hato | El capataz ingresa la información en el módulo correspondiente | El sistema guarda todos los datos generales del animal |
| 2.0 | Consultar ficha individual | El animal ya está registrado | El capataz busca y selecciona el animal | El sistema muestra la ficha completa del animal |
| 3.0 | Ver trazabilidad completa | El animal tiene historial de eventos | El capataz ingresa a la sección de trazabilidad | El sistema muestra historial sanitario, reproductivo, productivo y movimientos |
| 4.0 | Editar información | El animal requiere una actualización de datos | El capataz accede a la ficha y edita los datos | El sistema guarda las modificaciones realizadas |
