# HU015 — Quiero revisar auditorías del sistema

**Identificador:** HU015

## Historia de Usuario

> **Como** Administrador,  
> **quiero** revisar auditorías del sistema,  
> **para** conocer las acciones realizadas por los usuarios.

## Criterios de Aceptación

| # | Criterio | Contexto | Evento | Resultado Esperado |
|---|----------|----------|--------|--------------------|
| 1.0 | Registrar fecha, usuario y acción | El sistema registra actividades internas | Se realiza una acción relevante (login, edición, creación, etc.) | El sistema guarda la entrada en el historial de auditoría |
| 2.0 | Filtrar actividades por usuario o tipo | Se requiere consultar acciones específicas | El administrador aplica filtros en la vista de auditoría | El sistema muestra solo los resultados filtrados |
| 3.0 | Exportar auditoría | Se necesita documentar el historial | El administrador selecciona “Exportar auditoría” | El sistema genera el archivo con la información filtrada o completa |
