# HU006 — Quiero registrar control reproductivo

**Identificador:** HU006

## Historia de Usuario

> **Como** Veterinario,  
> **quiero** registrar control reproductivo,  
> **para** dar seguimiento a servicios, diagnósticos y partos..

## Criterios de Aceptación

| # | Criterio | Contexto | Evento | Resultado Esperado |
|---|----------|----------|--------|--------------------|
| 1.0 | Registrar servicios (monta/inseminación) | El animal está en etapa reproductiva | El veterinario ingresa el servicio realizado | El sistema almacena el servicio y calcula fechas estimadas |
| 2.0 | Registrar diagnósticos positivos o negativos | Se realiza diagnóstico reproductivo | El veterinario ingresa el resultado del diagnóstico | El sistema actualiza el estado reproductivo del animal |
| 3.0 | Registrar partos y generar ficha del ternero | La hembra ha tenido un parto | El veterinario registra el evento | El sistema registra el parto y genera automáticamente la ficha del nuevo ternero |
| 4.0 | Mostrar fechas estimadas de eventos | El animal tiene servicios registrados | El veterinario consulta el historial reproductivo | El sistema muestra próximas fechas probables de partos, revisiones o diagnósticos |
