# HU014 — Quiero recibir alertas y notificaciones

**Identificador:** HU014

## Historia de Usuario

> **Como** Usuario,  
> **quiero** recibir alertas y notificaciones,  
> **para** no olvidar eventos importantes.

## Criterios de Aceptación

| # | Criterio | Contexto | Evento | Resultado Esperado |
|---|----------|----------|--------|--------------------|
| 1.0 | Enviar notificaciones por correo o WhatsApp | El sistema detecta una fecha o evento próximo | El sistema genera la notificación automáticamente | El usuario recibe un mensaje por el canal configurado |
| 2.0 | Configurar tipo de alerta | El usuario desea personalizar sus notificaciones | El usuario modifica sus preferencias en el módulo de alertas | El sistema actualiza el tipo y frecuencia de las notificaciones |
| 3.0 | Mostrar historial de notificaciones | El usuario quiere revisar qué alertas se enviaron | El usuario ingresa a la vista de historial | El sistema muestra una lista con fecha, tipo y estado de las alertas |
