# HU011 — Quiero gestionar inventarios y compras de insumos

**Identificador:** HU011

## Historia de Usuario

> **Como** Administrador,  
> **quiero** gestionar inventarios y compras de insumos,  
> **para** evitar faltantes y registrar costos.

## Criterios de Aceptación

| # | Criterio | Contexto | Evento | Resultado Esperado |
|---|----------|----------|--------|--------------------|
| 1.0 | Registrar entradas y salidas de insumos | Se reciben o consumen insumos | El administrador ingresa la entrada o salida | El sistema actualiza las cantidades del inventario |
| 2.0 | Registrar compra de medicamentos, suplementos, etc. | Se adquiere algún insumo | El administrador registra la compra | El sistema almacena el gasto y entrada de insumos |
| 3.0 | Actualizar inventario automáticamente | Un insumo se usa en una actividad | El sistema detecta el consumo desde otros módulos | El inventario disminuye o aumenta según corresponda |
| 4.0 | Definir stock mínimo | Se quiere evitar quiebres de inventario | El administrador configura el mínimo de cada insumo | El sistema alerta cuando el stock es bajo |
