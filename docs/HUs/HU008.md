# HU008 — Quiero planificar y registrar actividades sanitarias

**Identificador:** HU008

## Historia de Usuario

> **Como** Veterinario,  
> **quiero** planificar y registrar actividades sanitarias,  
> **para** asegurar salud preventiva y correctiva en el hato.

## Criterios de Aceptación

| # | Criterio | Contexto | Evento | Resultado Esperado |
|---|----------|----------|--------|--------------------|
| 1.0 | Registrar tratamientos y enfermedades | Un animal presenta algún tipo de condición sanitaria | El veterinario registra el tratamiento o enfermedad detectada | El sistema almacena la información y la incorpora al historial sanitario |
| 2.0 | Crear planes sanitarios | Se requiere un plan de vacunación, desparasitación u otro | El veterinario define un plan sanitario | El sistema registra el plan y define fechas de aplicación |
| 3.0 | Ver calendario de actividades sanitarias | Existen actividades programadas | El veterinario consulta el calendario | El sistema muestra las actividades organizadas por fecha |
| 4.0 | Generar alertas de fechas próximas | Se acercan fechas importantes del plan | El sistema detecta eventos próximos | El sistema genera alertas automáticas para evitar olvidos |
