# HU010 — Quiero registrar información económica

**Identificador:** HU010

## Historia de Usuario

> **Como** Administrador,  
> **quiero** registrar información económica,  
> **para** evaluar la rentabilidad de la finca.

## Criterios de Aceptación

| # | Criterio | Contexto | Evento | Resultado Esperado |
|---|----------|----------|--------|--------------------|
| 1.0 | Registrar ingresos y egresos | Se realizan actividades económicas | El administrador ingresa los movimientos financieros | El sistema registra la operación y la clasifica |
| 2.0 | Asociar movimientos a actividades productivas | Un ingreso o egreso proviene de una actividad específica | El administrador selecciona la actividad relacionada | El sistema vincula los datos económicos a la producción correspondiente |
| 3.0 | Ver indicadores económicos básicos | Hay registros financieros en el sistema | El administrador accede al módulo de indicadores | El sistema muestra indicadores como costos, ganancias, balance y promedios |
