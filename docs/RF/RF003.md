# RF003 — Registrar lotes en una finca

| Campo | Valor |
|-------|-------|
| **Código** | RF003 |
| **Nombre** | Registrar lotes en una finca |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

El sistema debe permitir que el administrador registre los lotes dentro de cada finca. Cada lote debe contener información detallada como: nombre o código del lote, área (hectareas o metros), tipo de uso (cría, engorde, pastoreo, producción de leche), capacidad máxima de animales, y ubicación dentro de la finca. El registro de lotes permite organizar y gestionar de manera eficiente la distribución del ganado y los recursos de la finca. Cada lote estará asociado exclusivamente a una finca, y la finca debe estar previamente registrada. El administrador podrá crear, modificar o eliminar lotes según sea necesario, y cada acción quedará registrada en el historial de la finca para trazabilidad.

## Controles y Restricciones

- Control de no duplicidad: No se deben registrar dos lotes con el mismo nombre dentro de la misma finca.
- Control de obligatoriedad: Todos los campos obligatorios (nombre/código, área, tipo de uso) deben estar completos antes de guardar el lote.
- Control de capacidad: La capacidad máxima de animales por lote debe ser un valor positivo y no superar la capacidad total de la finca.
- Restricción de permisos: Solo el administrador puede crear o eliminar lotes.
- Restricciob de Permisos de Empleados: El administrador puede otorgar el permiso de modificar informaciòn relevante de un lote.

## Criterios de Aceptación

- El sistema valida que no existan lotes duplicados por nombre dentro de la misma finca.
- El administrador recibe confirmación en pantalla de que el lote fue registrado correctamente.
- Cada lote queda vinculado automáticamente a la finca correspondiente.
- Si falta información obligatoria, el sistema muestra un mensaje de error y no permite guardar el lote.
- El sistema verifica que la capacidad del lote sea coherente con la capacidad total de la finca.
- El historial de la finca registra automáticamente la creación, modificación o eliminación de lotes.
