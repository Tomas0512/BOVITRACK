# RF019 — Registro de Manejo de Lotes

| Campo | Valor |
|-------|-------|
| **Código** | RF019 |
| **Nombre** | Registro de Manejo de Lotes |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

El sistema debe permitir que el capataz, empleado o el administrador registren y gestionen la asignación y movimientos de los lotes de animales dentro de la finca, incluyendo: Identificación del lote y tipo de animales que contiene (terneros, novillas, vacas aptas para reproducción, vacas de leche, ganado de engorde, etc.). Fecha de ingreso y salida de cada lote a un potrero o sector de la finca. Cantidad de animales en el lote y características relevantes (edad promedio, peso promedio, sexo, raza). Observaciones sobre comportamiento, desarrollo, salud o necesidades del lote. Movimientos de animales individuales dentro del lote o entre lotes (traslado por crecimiento, reproducción o cuarentena). Este registro permite controlar la ubicación, desarrollo y bienestar de cada lote, planificar la alimentación, reproducción y destete, y garantizar trazabilidad de todos los animales.

## Controles y Restricciones

- Control de registro por lote: Cada lote debe estar previamente registrado en el sistema.
- Control de consistencia: La fecha de ingreso y salida debe ser coherente con el historial del lote y de los animales que lo componen.
- Control de capacidad: No se pueden asignar más animales a un lote que la capacidad máxima permitida del potrero.
- Control de seguimiento: Los movimientos de animales individuales deben actualizar automáticamente el lote de origen y el de destino.
- Control de fecha/hora: Cada registro de movimiento debe incluir fecha y hora para trazabilidad.
- Restricción de permisos: Solo el administrador, capataz o empleado pueden registrar o modificar lotes; usuarios sin permisos no pueden acceder.
- Restricción de datos obligatorios: No se permite guardar registros incompletos, como identificación de lote, fecha de ingreso/salida o cantidad de animales.

## Criterios de Aceptación

- El sistema permite registrar únicamente lotes válidos previamente creados.
- Se notifican errores si faltan campos obligatorios o los valores ingresados no son válidos.
- Cada registro incluye fecha, hora, usuario que realizó el registro, cantidad de animales y observaciones.
- Los reportes muestran correctamente la ubicación y estado de cada lote dentro de la finca.
- Se puede generar un historial completo de cada lote para trazabilidad y planificación de actividades.
- Los movimientos de animales individuales actualizan automáticamente el lote de origen y destino.
