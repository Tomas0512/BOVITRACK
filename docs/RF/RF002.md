# RF002 — Registrar una finca

| Campo | Valor |
|-------|-------|
| **Código** | RF002 |
| **Nombre** | Registrar una finca |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

El sistema debe permitir que el administrador o dueño registre una finca en la plataforma. Durante este proceso, el administrador será responsable de ingresar información detallada y obligatoria como: nombre de la finca, Direccion, departamento, como datos opcionales(ciudad, municipio),  área total( Hectarias o metros cuatrados ), propósito productivo (carne, leche o doble propósito), numero de identificacion de la finca y datos de contacto(Numero de telefono de la Finca). El administrador será el único con la capacidad de crear nuevas fincas y asociarlas a su cuenta de usuario. Además, se debe registrar automáticamente la fecha de creación de la finca en el sistema, y cada finca quedará vinculada de forma exclusiva al administrador que la registró. Una finca no puede existir sin un administrador asignado.

## Controles y Restricciones

- Control de no duplicidad: No se debe permitir registrar dos fincas con el mismo nombre y la misma ubicación geográfica.
- Control de obligatoriedad: Todos los campos obligatorios (nombre, ubicación, propósito, numero de identificación de la finca) deben estar diligenciados para continuar.
- Restricción de permisos: Solo el administrador podrá registrar nuevas fincas; empleados o capataces no tendrán acceso a esta función.
- Restricción de integridad: Una finca debe estar siempre asociada a un administrador, no puede existir de manera independiente en el sistema.

## Criterios de Aceptación

- El sistema valida que el nombre de la finca y ubicación no estén duplicados antes de guardarla.
- Se notifica al administrador mediante un mensaje en pantalla que la finca fue registrada correctamente.
- La finca queda vinculada automáticamente al usuario administrador que la creó.
- El sistema muestra la finca recién creada en la lista de fincas disponibles del administrador.
- Si falta información obligatoria, el sistema muestra un mensaje de error y no permite continuar con el registro.
