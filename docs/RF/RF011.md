# RF011 — Registro de Ordeño y Producción de Leche

| Campo | Valor |
|-------|-------|
| **Código** | RF011 |
| **Nombre** | Registro de Ordeño y Producción de Leche |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

El sistema debe permitir que el capataz o empleado registre diariamente la producción de leche de cada vaca o lote de vacas, incluyendo: Identificación de la vaca o del lote. Fecha y hora del ordeño. Cantidad de leche producida (en litros o kilos). Tipo de ordeño (manual o mecánico). Observaciones sobre comportamiento, salud o anomalías durante el ordeño. El registro permitirá llevar un seguimiento de la producción, calcular la producción promedio diaria por vaca o lote y permitir análisis de eficiencia y calidad del ordeño.

## Controles y Restricciones

- Control de registro por vaca o lote: Cada registro debe estar vinculado a una vaca o lote previamente registrado en el sistema.Control de cantidades: La cantidad de leche registrada debe ser un número positivo y coherente con la producción esperada por vaca o lote.Control de consistencia: El sistema puede alertar si la producción está fuera de los rangos promedio históricos para esa vaca o lote.
- Control de fecha/hora: Cada registro debe incluir fecha y hora para trazabilidad.Restricción de permisos: Solo empleados autorizados o el administrador pueden registrar ordeño; usuarios sin permisos no pueden acceder.
- Restricción de datos obligatorios: No se permite guardar registros incompletos, como cantidad, fecha/hora o identificación de vaca/lote.

## Criterios de Aceptación

- El sistema permite registrar ordeño únicamente para vacas o lotes válidos.
- Se notifican errores si faltan campos obligatorios o los valores ingresados no son válidos.
- Cada registro incluye fecha, hora, usuario que realizó el registro y cantidad de leche producida.
- Los reportes muestran correctamente la producción diaria, semanal y mensual por vaca o lote.
- El historial de cada vaca o lote registra automáticamente todos los ordeños para trazabilidad.
- El sistema puede calcular la producción promedio y alertar si hay disminución significativa.
