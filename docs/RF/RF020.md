# RF020 — Registro de Pesaje y Desarrollo de Ganado de Engorde

| Campo | Valor |
|-------|-------|

| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

El sistema debe permitir que el capataz, empleado o el administrador registre los pesajes y el seguimiento del desarrollo de los animales destinados a engorde, incluyendo: Identificación del animal o lote. Fecha y hora del pesaje. Peso actual del animal y variación respecto al pesaje anterior. Tipo de alimentación recibida (pasto, concentrado, silo, suplementos). Observaciones sobre salud, comportamiento y desarrollo del animal o lote. Este registro permite monitorear la ganancia de peso, evaluar la eficiencia alimentaria y planificar la venta o traslado de animales según su desarrollo y metas de producción.

## Controles y Restricciones

- Control de registro por animal o lote: Cada pesaje debe estar vinculado a animales o lotes previamente registrados.
- Control de consistencia: Las fechas y pesos deben ser coherentes con el historial del animal o lote.
- Control de variación de peso: El sistema alerta si la ganancia o pérdida de peso excede los rangos normales definidos.
- Control de fecha/hora: Cada registro debe incluir fecha y hora para trazabilidad.
- Restricción de permisos: Solo el administrador, capataz o empleados autorizados pueden registrar pesajes.
- Restricción de datos obligatorios: No se permite guardar registros incompletos, como identificación de animal/lote, fecha o peso.

## Criterios de Aceptación

- El sistema permite registrar pesajes únicamente para animales o lotes válidos.
- Se notifican errores si faltan campos obligatorios o los valores ingresados no son válidos.
- Cada registro incluye fecha, hora, usuario que realizó el registro, peso y observaciones.
- Los reportes muestran correctamente la evolución de peso de cada animal o lote por período.
- Se puede generar un historial completo del desarrollo del animal o lote para planificación de venta o traslado.
- El sistema alerta si se detectan anomalías en la ganancia de peso o en la alimentación registrada.
