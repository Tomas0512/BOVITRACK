# RF014 — Registro de Nacimientos de Terneros

| Campo | Valor |
|-------|-------|
| **Código** | RF014 |
| **Nombre** | Registro de Nacimientos de Terneros |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

El sistema debe permitir que el capataz, empleado o el administrador registre todos los eventos relacionados con el nacimiento de un ternero, incluyendo: Identificación de la madre y del ternero. Fecha y hora del nacimiento. Datos básicos del ternero: padre, madre, sexo, peso, color, señales y observaciones físicas. Procedimientos realizados al nacimiento: tatuaje, descorne, curación de ombligo, administración de vitaminas y calcio. Registro de la alimentación inicial: calostro, leche materna o suplementaria según el protocolo de la finca.

## Controles y Restricciones

- Control de registro por ternero y madre: Cada nacimiento debe estar vinculado a una vaca madre previamente registrada.
- Control de consistencia: No se pueden registrar nacimientos con fechas futuras o inconsistentes.
- Control de procedimientos: Solo se pueden seleccionar procedimientos válidos definidos en el protocolo de la finca.
- Control de alimentación inicial: Debe registrarse obligatoriamente si el ternero recibió calostro, leche materna o suplementaria.
- Control de fecha/hora: Cada registro debe incluir fecha y hora exacta del evento para trazabilidad.
- Restricción de permisos: Solo el administrador, capataz o empleados autorizados pueden registrar nacimientos.
- Restricción de datos obligatorios: No se permite guardar registros incompletos (madre, ternero, fecha, hora y peso son mínimos requeridos).

## Criterios de Aceptación

- El sistema permite registrar nacimientos únicamente para vacas y terneros válidos.
- Se notifican errores si faltan campos obligatorios o los valores ingresados no son válidos.
- Cada registro incluye fecha, hora y usuario que realizó el registro.
- Los reportes muestran correctamente los nacimientos por lote, madre o fecha.
- Se puede generar un historial individual de cada ternero desde su nacimiento.
