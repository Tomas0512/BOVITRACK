# RF001 — Registrar un Administrador/Dueño

| Campo | Valor |
|-------|-------|
| **Código** | RF001 |
| **Nombre** | Registrar un Administrador/Dueño |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

El sistema debe permitir que una persona pueda registrarse como Administrador/Dueño de finca para tener control total sobre la información y usuarios asociados a su finca. Durante el proceso de registro, el administrador debe proporcionar datos personales obligatorios como nombres, apellidos, tipo y número de documento de identidad, correo electrónico, número de teléfono, una contraseña segura y verificar la contraseña.

## Controles y Restricciones

- Control registro nombre,apellido:El sistema debe permitir solo texto al momento de registrar el nombre y apellido,Control registro telefonico:El sistema debe permitir solo valores numericos, Control de correo electrónico: El sistema debe validar que no exista otro administrador registrado con el mismo correo y correcta inscripcion de formato.
- Control de documento de identidad: El sistema debe verificar que el documento sea único,numerico y no esté asociado a otra cuenta.
- Control de contraseña: La contraseña debe cumplir con los parámetros de seguridad establecidos (mínimo 8 caracteres, incluir 1 mayúsculas, 1 minúsculas, números y 1 símbolo especial) y debe coincidir al verificarla.Control de validación de correo: El sistema debe enviar automáticamente un enlace de verificación y no permitir el acceso hasta que se confirme.Restricción de acceso: Solo los administradores pueden registrarse de forma autónoma; los empleados dependen del administrador para la creación de su usuario.
- Restricción de datos incompletos: No se podrá finalizar el registro si faltan datos obligatorios.
- Restricción de confirmación: El administrador no podrá usar el sistema hasta verificar su correo electrónico.
- Restricción de Autorización: El administrador no se podra registrar hasta autorizar el tratamiento de sus datos, Restricción de terminos y condiciones: El administrador no se podra registrar hasta no aceptar los términos y condiciones

## Criterios de Aceptación

- El sistema impide registrar un administrador con un correo ya existente.
- El sistema impide registrar un administrador con un documento duplicado.
- El sistema envía un correo de verificación tras completar el registro.
- La cuenta permanece inactiva hasta que el administrador confirme el correo.
- El sistema valida en tiempo real la fortaleza de la contraseña.
- Se muestran mensajes claros en caso de errores (correo duplicado, documento inválido, etc.).
- Una vez confirmado el correo, el administrador puede iniciar sesión y registrar su finca.
