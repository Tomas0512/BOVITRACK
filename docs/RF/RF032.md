# RF032 — Notificaciones por Correo o Whatsapp

| Campo | Valor |
|-------|-------|
| **Código** | RF032 |
| **Nombre** | Notificaciones por Correo o Whatsapp |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

El sistema debe enviar notificaciones por correo o WhatsApp a los usuarios.
Estas notificaciones pueden ser alertas (vacunas, partos, tratamientos), recordatorios o resúmenes semanales de producción y gastos.

## Controles y Restricciones

- Control de acceso: Cada usuario debe tener configurado su correo o número de WhatsApp.
- Control de consentimiento: El sistema valida que el usuario haya autorizado recibir notificaciones.
- Control de tiempo: Las notificaciones se envían en el horario definido (ejemplo: horario laboral).
- Restricción de permisos: Solo el administrador puede configurar qué notificaciones se envían.
- Restricción de acceso: Los empleados solo reciben notificaciones de las fincas donde trabajan.
- Restricción funcional: No se envían notificaciones si no hay datos que las respalden.
- Restricción de seguridad: Los mensajes deben enviarse de forma segura para proteger la información personal.

## Criterios de Aceptación

- Las alertas aparecen en pantalla y, si está configurado, se envían también al correo o WhatsApp.
- El usuario puede marcar notificaciones como recibidas o darse de baja de cierto tipo de avisos.
- El sistema guarda un registro de todas las notificaciones enviadas con fecha, destinatario y tipo.
- Si un envío falla, el sistema lo registra y muestra un aviso al administrador.
- Los resúmenes semanales incluyen los indicadores principales de la finca (producción, gastos, eventos próximos).
