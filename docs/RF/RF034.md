# RF034 — Gestión de Compras de Insumos

| Campo | Valor |
|-------|-------|
| **Código** | RF034 |
| **Nombre** | Gestión de Compras de Insumos |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

El sistema debe permitir registrar todas las compras de insumos de la finca (alimentos, medicamentos, suplementos, herramientas), incluyendo: identificación del proveedor, tipo de insumo, cantidad adquirida, unidad de medida, costo unitario, costo total, fecha de compra, fecha de vencimiento (si aplica), número de factura y observaciones.

## Controles y Restricciones

- Control de obligatoriedad: Cada compra debe registrar proveedor, insumo, cantidad, costo y fecha.
- Control de valores: No se podrán registrar cantidades o montos negativos.
- Control de stock: Al registrar la compra, el inventario de insumos debe actualizarse automáticamente.
- Restricción de permisos: Solo el administrador o empleados autorizados podrán registrar compras.
- Restricción documental: Para compras de medicamentos, debe adjuntarse documento soporte (ejemplo: receta o factura).

## Criterios de Aceptación

- Cada compra registrada actualiza automáticamente el inventario.
- El sistema alerta si la fecha de vencimiento ingresada es anterior a la fecha actual.
- Los reportes económicos reflejan inmediatamente el gasto asociado a la compra.
- El historial de compras por proveedor e insumo puede consultarse en cualquier momento.
