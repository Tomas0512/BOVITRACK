# RF018 — Planificacion y Control Sanitario

| Campo | Valor |
|-------|-------|
| **Código** | RF018 |
| **Nombre** | Planificacion y Control Sanitario |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

El sistema debe permitir planificar, controlar y dar seguimiento a las aplicaciones sanitarias en el hato. Esto incluye el control de stock de medicamentos y vacunas, evitando que se registren aplicaciones si no hay existencias suficientes. Asimismo, el sistema debe generar un calendario sanitario con fechas programadas para futuras vacunaciones, desparasitaciones y suplementaciones, enviando alertas cuando algún animal o lote no ha recibido la aplicación correspondiente según el protocolo sanitario de la finca o las normativas gubernamentales.

## Controles y Restricciones

- Control de stock: El sistema no permite registrar aplicaciones de vacunas o medicamentos si no hay stock suficiente.
- Control de fechas: Las aplicaciones planificadas deben corresponder a intervalos definidos por protocolos sanitarios.
- Control de alertas: El sistema debe notificar automáticamente cuando un animal o lote esté próximo o haya superado la fecha de aplicación.
- Restricción de permisos: Solo el administrador o empleados autorizados pueden modificar el stock y configurar el calendario sanitario.

## Criterios de Aceptación

- El sistema alerta si algún animal o lote no ha recibido la aplicación correspondiente según protocolo.
- Se puede generar un calendario sanitario por animal o lote, con próximas fechas de vacunación y desparasitación.
- El sistema bloquea el registro de aplicaciones si no hay suficiente stock.
- Los reportes muestran consumo de medicamentos, vacunas y suplementos, comparando con el stock disponible.
