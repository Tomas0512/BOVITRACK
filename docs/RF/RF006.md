# RF006 — Recuperación de Cuentas

| Campo | Valor |
|-------|-------|
| **Código** | RF006 |
| **Nombre** | Recuperación de Cuentas |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

El sistema debe permitir a los usuarios recuperar el acceso a su cuenta en caso de olvido de contraseña o pérdida de credenciales. El proceso debe realizarse de manera segura mediante. Solicitud de restablecimiento desde la pantalla de inicio de sesión. Envío de un enlace seguro al correo electrónico registrado del usuario. Validación de vigencia del enlace (ejemplo: 30 minutos).

## Controles y Restricciones

- Control de correo válido: Solo se enviará el enlace de recuperación a correos electrónicos registrados en el sistema.
- Control de unicidad: Un enlace de recuperación solo puede usarse una vez.
- Restricción de seguridad: No se permite reutilizar contraseñas anteriores.
- Restricción de vigencia: El enlace expira automáticamente después del tiempo definido.
- Restricción de obligatoriedad: El usuario debe completar todos los pasos (enlace, validación, nueva contraseña).

## Criterios de Aceptación

- Si el usuario ingresa un correo no registrado, el sistema debe mostrar un mensaje claro sin revelar información sensible.
- El sistema envía un enlace seguro al correo válido en menos de 1 minuto.
- El usuario puede cambiar su contraseña exitosamente desde el enlace, y esta entra en vigor de inmediato.
- El historial registra la fecha, hora y usuario que realizó el restablecimiento.
