# RF004 — Administracion de Empleados

| Campo | Valor |
|-------|-------|
| **Código** | RF004 |
| **Nombre** | Administracion de Empleados |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

El sistema debe permitir que el administrador registre nuevos empleados asociados a su finca. Durante el registro, se deben ingresar los siguientes datos obligatorios: nombres , Apellidos , tipo y número de documento de identidad, correo electrónico, número de teléfono , contraseña y la finca a la que se va a asociar. El administrador será el único responsable de crear, editar y/o eliminar estos usuarios, asignarles permisos según su rol y vincularlos a la finca correspondiente. Cada empleado tendrá acceso restringido al sistema, limitado a las funciones que le corresponden según su rol, sin poder crear o modificar usuarios o fincas y solo si es autorizado podra modificar lotes.

## Controles y Restricciones

- Control de correo electrónico: El sistema valida que el correo del empleado no esté duplicado.Control de documento de identidad: El sistema verifica que no exista otro empleado con el mismo documento.
- Control de rol: El administrador asigna el rol correcto; el sistema restringe funciones según rol.
- Control de asociación a finca: Cada empleado debe estar vinculado obligatoriamente a una finca registrada.Restricción de permisos: Solo el administrador puede registrar, modificar o desactivar usuarios; los empleados no pueden crear cuentas por sí mismos.Restricción de datos obligatorios: No se permite guardar el registro si faltan datos esenciales.

## Criterios de Aceptación

- El sistema bloquea el registro de un empleado con correo duplicado.
- El sistema bloquea el registro de un empleado con documento de identidad duplicado.
- El sistema notifica al administrador cuando un empleado ha sido registrado correctamente.
- El empleado recibe acceso solo a las funciones permitidas según su rol.
- Cada empleado queda asociado automáticamente a la finca indicada.
- El historial registra la creación, modificación o eliminación de empleados.
