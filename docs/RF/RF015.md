# RF015 — Registro de Cria y Desarrollo de Terneros

| Campo | Valor |
|-------|-------|
| **Código** | RF015 |
| **Nombre** | Registro de Cria y Desarrollo de Terneros |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

El sistema debe permitir que el capataz, empleado o el administrador registre y consulte la información asociada a la cría y desarrollo de los terneros, incluyendo: Fecha estimada y real de destete. Tipo de alimentación durante la transición (leche, concentrado, pasto). Observaciones sobre salud, desarrollo y comportamiento del ternero. El sistema debe permitir que el capataz, empleado o el administrador registre todos los eventos relacionados con el nacimiento de un ternero, incluyendo: Identificación de la madre y del ternero. Fecha y hora del nacimiento. Datos básicos del ternero: padre, madre, sexo, peso, color, señales y observaciones físicas. Procedimientos realizados al nacimiento: tatuaje, descorne, curación de ombligo, administración de vitaminas y calcio. Registro de la alimentación inicial: calostro, leche materna o suplementaria según el protocolo de la finca.

## Controles y Restricciones

- Control de vínculo: La cría debe estar asociada a un ternero previamente registrado en el sistema.
- Control de consistencia: No se pueden registrar destetes antes de la fecha de nacimiento.
- Control de alimentación: Debe registrarse obligatoriamente el tipo de alimentación durante la etapa de transición.
- Control de fecha/hora: Cada evento de cría debe incluir fecha y hora para trazabilidad.
- Restricción de permisos: Solo el administrador, capataz o empleados autorizados pueden registrar o modificar la información de cría.
- Restricción de datos obligatorios: No se permite guardar registros de cría sin fecha de destete ni tipo de alimentación.

## Criterios de Aceptación

- El sistema permite registrar eventos de cría únicamente para terneros previamente registrados.
- Se notifican errores si faltan campos obligatorios o los valores ingresados no son válidos.
- Cada registro de cría incluye fecha, hora y usuario que lo realizó.
- Los reportes muestran correctamente el crecimiento, alimentación y destete de los terneros.
- Se puede generar un historial completo del desarrollo de cada ternero para seguimiento de salud y planificación de alimentación y destete.
