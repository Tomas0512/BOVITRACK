# RF008 — Iniciar Sesión

| Campo | Valor |
|-------|-------|
| **Código** | RF008 |
| **Nombre** | Iniciar Sesión |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

El sistema debe permitir que todos los usuarios registrados (administradores, empleados y capataces) puedan iniciar sesión de manera segura usando su correo electrónico o numero de documento y contraseña. Dependiendo del rol del usuario, el sistema mostrará el panel y las funciones disponibles correspondientes: Administrador: acceso completo a fincas, lotes, empleados, registros productivos, económicos y sanitarios. Capataz/Empleado: acceso limitado a tareas asignadas, como registrar animales, alimentación, ordeño, sanidad o producción según sus permisos. El sistema debe garantizar la seguridad de la información y mantener los datos de cada usuario privados y confidenciales.

## Controles y Restricciones

- Control de autenticación: Verifica que el correo y la contraseña coincidan con los registros del sistema.
- Control de rol: Determina las funcionalidades que se muestran según el rol del usuario.
- Control de intentos fallidos: Bloquea temporalmente la cuenta si se exceden intentos de acceso incorrectos (por ejemplo, 5 intentos).
- Restricción de acceso: Solo usuarios registrados y confirmados por correo pueden iniciar sesión.Restricción de seguridad: La contraseña debe transmitirse y almacenarse de manera segura (cifrada).
- Restricción de sesión activa: Cada usuario solo puede iniciar sesión en dispositivos autorizados según política de seguridad.

## Criterios de Aceptación

- El sistema permite iniciar sesión únicamente con credenciales válidas.
- El sistema bloquea el acceso tras múltiples intentos fallidos y notifica al usuario.
- Los usuarios solo ven las funcionalidades permitidas según su rol.
- Se registra automáticamente el inicio de sesión y la fecha/hora para auditoría.
- El sistema protege la contraseña mediante cifrado seguro.
- Si un usuario no ha confirmado su correo, no puede iniciar sesión.
