# RF012 — Registro de Servicios Reproductivos

| Campo | Valor |
|-------|-------|
| **Código** | RF012 |
| **Nombre** | Registro de Servicios Reproductivos |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

El sistema debe permitir que el capataz, empleado o administrador registre todos los servicios reproductivos realizados a las vacas, incluyendo: Identificación de la vaca y, si aplica, del toro. Tipo de servicio reproductivo: monta natural o inseminación artificial. Fecha y hora del servicio o inseminación. Observaciones sobre salud, comportamiento reproductivo o anomalías. Este registro permitirá llevar el control de los servicios realizados, la planificación de inseminaciones futuras y el cálculo de indicadores reproductivos.

## Controles y Restricciones

- Control de registro por vaca: Cada servicio debe estar vinculado a una vaca previamente registrada.
- Control de consistencia: No se pueden registrar servicios en fechas incoherentes (ejemplo: fecha futura o antes del nacimiento de la vaca).
- Control de procedimientos: Solo se permiten los tipos de servicio definidos (monta natural o inseminación artificial).
- Control de fecha/hora: Cada registro debe incluir fecha y hora exacta para trazabilidad.
- Restricción de permisos: Solo empleados autorizados o el administrador pueden registrar servicios reproductivos.
- Restricción de datos obligatorios: No se permite guardar registros incompletos (tipo de servicio, fecha, identificación de la vaca).

## Criterios de Aceptación

- El sistema permite registrar servicios reproductivos únicamente para vacas válidas.
- Se notifican errores si faltan campos obligatorios o los valores ingresados no son válidos.
- Cada registro incluye fecha, hora y usuario que realizó el registro.
- Los reportes muestran correctamente los servicios reproductivos por vaca, lote y período.
- Se puede generar un historial de servicios por cada vaca para análisis reproductivo.
