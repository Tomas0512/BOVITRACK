# RF035 — Gestión de Usuarios Externos

| Campo | Valor |
|-------|-------|
| **Código** | RF035 |
| **Nombre** | Gestión de Usuarios Externos |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

El sistema debe permitir registrar usuarios externos como veterinarios o consultores, asignándoles permisos limitados para visualizar o registrar información específica (sanidad, reproducción, reportes técnicos). Estos usuarios externos no tendrán acceso a información económica ni administrativa de la finca.

## Controles y Restricciones

- Control de obligatoriedad: Cada usuario externo debe registrarse con nombre, correo electrónico y rol definido.
- Control de permisos: Los permisos de usuarios externos se limitan a los módulos definidos por el administrador.
- Restricción de acceso: Un usuario externo solo puede acceder a la finca a la que fue asignado.
- Restricción de confidencialidad: Los usuarios externos no podrán visualizar datos financieros o de administración de empleados.

## Criterios de Aceptación

- Al crear un usuario externo, el sistema valida que se le asigne un rol y permisos mínimos de acceso.
- Los usuarios externos solo pueden acceder a la información autorizada por el administrador.
- El historial registra todas las acciones realizadas por usuarios externos con fecha, hora y acción.
- El sistema bloquea automáticamente intentos de acceso no autorizados por parte de usuarios externos.
