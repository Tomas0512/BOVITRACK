# RF013 — Registro de Resultados Reproductivos y Seguimiento de Embarazo

| Campo | Valor |
|-------|-------|
| **Código** | RF013 |
| **Nombre** | Registro de Resultados Reproductivos y Seguimiento de Embarazo |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

Control de consistencia: No se puede registrar un parto antes de la fecha estimada de concepción.    Control de obligatoriedad: Cada registro debe incluir al menos la identificación de la vaca, resultado del servicio y fecha.    Control de resultados válidos: Solo se admiten los valores preñada, no preñada o aborto.    Restricción de permisos: Solo capataz, empleados autorizados o el administrador pueden registrar resultados.     Restricción de integridad: Los registros deben estar vinculados a un servicio reproductivo previamente registrado.

## Controles y Restricciones

- El sistema notifica errores si faltan datos obligatorios o si las fechas ingresadas no son coherentes.
- Cada registro queda trazado con usuario, fecha y hora de creación.
- El historial reproductivo de la vaca se actualiza automáticamente con el resultado y seguimiento del embarazo.
- Se pueden generar reportes con tasas de concepción, intervalos entre partos y porcentaje de abortos.
- El sistema genera un calendario con las fechas estimadas de parto para facilitar la gestión.

## Criterios de Aceptación

- El sistema permite registrar servicios reproductivos únicamente para vacas válidas.
- Se notifican errores si faltan campos obligatorios o los valores ingresados no son válidos.
- Cada registro incluye fecha, hora, usuario que realizó el registro y resultado del servicio.
- Los reportes muestran indicadores reproductivos correctamente calculados por vaca, lote y período.
- El historial reproductivo de cada vaca permite planificar inseminaciones futuras y seguimiento de partos.
- Se puede generar un calendario de servicios y fechas estimadas de parto para gestión de la finca.
