# RNF0010 — Permitir personalizacion visual

| Campo | Valor |
|-------|-------|
| **Código** | RNF0010 |
| **Nombre** | Permitir personalizacion visual |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

La interfaz debe permitir cambios de colores, logotipos y elementos de estilo para cada finca o cliente.

## Controles y Restricciones

Mantener separación entre la lógica del negocio y las plantillas de diseño; soporte de hojas de estilo configurables.

## Criterios de Aceptación

Un usuario puede cambiar colores, logotipo o tema y ver los cambios reflejados sin afectar las funciones.
