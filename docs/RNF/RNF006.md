# RNF006 — Facilitar el uso con una interfaz intuitiva, adaptable a móviles

| Campo | Valor |
|-------|-------|
| **Código** | RNF006 |
| **Nombre** | Facilitar el uso con una interfaz intuitiva, adaptable a móviles |
| **Versión** | 3.0 |

## Descripción

La interfaz debe ser intuitiva y adaptable.

## Controles y Restricciones

Compatible con dispositivos móviles y navegadores modernos.

## Criterios de Aceptación

Los usuarios logran registrar información sin capacitación extensa.
