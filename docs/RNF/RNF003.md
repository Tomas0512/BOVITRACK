# RNF003 — Mantener disponibilidad 24/7

| Campo | Valor |
|-------|-------|
| **Código** | RNF003 |
| **Nombre** | Mantener disponibilidad 24/7 |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

El sistema debe estar disponible para uso continuo.

## Controles y Restricciones

Disponibilidad mínima del 99.5 % mensual.

## Criterios de Aceptación

El monitoreo demuestra cumplimiento del SLA de disponibilidad.
