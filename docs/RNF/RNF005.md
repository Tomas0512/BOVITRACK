# RNF005 — Realizar copias de seguridad diarias

| Campo | Valor |
|-------|-------|
| **Código** | RNF005 |
| **Nombre** | Realizar copias de seguridad diarias |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

El sistema debe garantizar respaldo de la información.

## Controles y Restricciones

- Copias de seguridad automáticas al menos una vez al día.
- Restauración en menos de X horas.

## Criterios de Aceptación

Se realizan copias diarias verificables y restauración exitosa en pruebas.
