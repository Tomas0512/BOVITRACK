# RNF008 — Permitir recuperación rápida

| Campo | Valor |
|-------|-------|
| **Código** | RNF008 |
| **Nombre** | Permitir recuperación rápida |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

El sistema debe restaurar la información y reanudar operaciones en caso de falla o desastre.

## Controles y Restricciones

Contar con planes de contingencia y pruebas de recuperación; tiempo máximo de restauración definido (por ejemplo, 4 horas).

## Criterios de Aceptación

En simulacros de caída total, la información se recupera.
