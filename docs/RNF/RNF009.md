# RNF009 — Permitir despliegue en la nube

| Campo | Valor |
|-------|-------|
| **Código** | RNF009 |
| **Nombre** | Permitir despliegue en la nube |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

El sistema debe poder instalarse y ejecutarse en plataformas de nube pública o privada.

## Controles y Restricciones

Uso de tecnologías portables y configuración que no dependa de hardware específico.

## Criterios de Aceptación

El sistema se implementa correctamente en un entorno de nube.
