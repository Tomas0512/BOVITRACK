# RNF004 — Cifrar y proteger los datos

| Campo | Valor |
|-------|-------|
| **Código** | RNF004 |
| **Nombre** | Cifrar y proteger los datos |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

Toda la información sensible debe estar protegida.

## Controles y Restricciones

Uso obligatorio de HTTPS y cifrado en base de datos para datos sensibles.

## Criterios de Aceptación

Se valida que los datos viajan cifrados y no se guardan en texto plano.
