# RNF002 — Proteger el acceso con inicio de sesión seguro y roles de usuario

| Campo | Valor |
|-------|-------|
| **Código** | RNF002 |
| **Nombre** | Proteger el acceso con inicio de sesión seguro y roles de usuario |
| **Versión** | 3.0 |
| **Prioridad** | Alta |

## Descripción

El sistema debe autenticar usuarios y manejar roles de acceso.

## Controles y Restricciones

- Uso de credenciales seguras y cifrado en login.
- Definición de roles (Administrador, Capataz, Empleado).

## Criterios de Aceptación

Un usuario no autorizado no puede acceder; roles acceden solo a lo permitido.
